🎮 Koleksi Game

Panduan Instalasi & Setup

✨ Project sudah siap! Ikuti langkah-langkah di bawah ini.

📋 Daftar Fitur

  • Dashboard dengan statistik real-time
  • Simpan dan kelola koleksi game
  • Buat wishlist dengan prioritas
  • Lacak status penyelesaian (tamat/belum)
  • Tracking progress per game
  • Kategori game yang terorganisir
  • Responsive design (mobile-friendly)
  • Database SQLite (file-based, no server)

🚀 Cara Menjalankan

  1. Buka Laragon
    C:\laragon\Laragon.exe
  2. Klik "Start All" untuk menjalankan Apache & PHP
  3. Akses aplikasi di browser:
    http://koleksigame.test/public/
    atau
    http://localhost/koleksigame/public/

📁 Struktur Project

koleksigame/
├── public/ ← Folder utama (akses dari browser)
│ ├── index.php ← Dashboard
│ ├── koleksi.php ← Koleksi game
│ └── wishlist.php ← Wishlist
├── includes/ ← Backend files
│ ├── config.php ← Database config
│ └── functions.php ← Functions
├── assets/ ← CSS & images
│ ├── css/ ← Bootstrap + style.css
│ └── images/ ← Gambar game
├── data/ ← Database (auto-created)
├── README.md ← Full documentation
└── schema.sql ← Database schema

💾 Database

Database otomatis dibuat saat pertama kali akses aplikasi:

Location: koleksigame/data/koleksigame.db
Type: SQLite (file-based)
Tables: users, game, kategori, koleksi, wishlist

🎯 Teknologi

  • Backend: PHP 7.0+
  • Database: SQLite
  • Frontend: HTML5 + Bootstrap 5
  • Styling: CSS3 + Bootstrap custom
